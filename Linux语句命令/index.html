Linux 语句命令

cd

返回上一级目录

cd ..

返回上上级目录

cd ../../

进入目录下指定目录

cd documents/file/

返回家目录

cd ~

返回上个所在目录

cd -

进入绝对路径

cd /root/*/*/

ls

展示当前目录文件(文件包括文件夹)

ls

检视目录文件全部信息(包括权限rwx,文件大小,文件时间信息)

ls -l

列出文件的同时查看文件的大小

ls -lh

显示全部文件包括隐藏文件

ls -a

ls帮助文档

ls --help

touch

创建文件,加入后缀更好识别,不加也可

touch file.py

同时创建多个文件

touch file2.py file3.py file4.py

mkdir

创建文件夹dir

mkdir dir/

创建dir目录以及在dir目录下创建dir2目录

mkdir dir/dir2/

rm

移除文件file

rm file

谨慎 移除目录下所以文件

rm *

询问是否移除(y/n)

rm -i *

逐句询问是否移除file1 file2 file3

rm -i file1 file2 file3

大于3个文件才会询问

rm -I *

递归删除dir文件夹(包括内部所以文件)

rm -r dir/

cp

复制文件。如果fileCopy已经存在,会被file复制内容重新覆盖

cp file.txt fileCopy.txt

问询是否复制(或者覆盖)(y/n)

cp -i file fileCopy

加入/符号,表示文件夹。将file复制到dir文件夹中

cp file dir/

-R递归,将dir/中所有文件复制到dir2/中, 不加-R会报错

cp -R dir/ dir2/

将file开头的文件,都复制到dir/目录中

cp file* dir/

将5结尾的文件,都复制到dir/目录中

cp *5 dir/

默认最后为复制到的目录, 前面为复制的文件

cp file1 file2 dir/

mv

file移动到dir/

mv file dir/

移动file到fileReName,相当于重命名

mv file fileReName

rmdir

仅可移除空文件夹

rmdir dir/

vim

vim 文本编辑器,可用于读写文件

vim file.py

cat

显示 a.py 内容,可用于读文件

cat a.py

将 a.py 数据放到 b.py 相当于复制

cat a.py > b.py

a.py b.py 数据合并放到 c.py

cat a.py b.py > c.py

将 a.py 数据补充放到 b.py 后面

cat a.py >> b.py

chmod

(u或与g或与a) (+或-) (r或与w或与x) file.py (用户 或与 组 或与 所有人) (憎加 或 减少) (读 或与 写 或与 执行) (目标) 权限 (d)(rwx)(r-x)(rwx) (-)(rwx)(-wx)(rw-) 有4个组成部分以括号隔开;d表示文件夹,-表示文件;r表示读;w表示写;x表示执行; 第1个部分文件类型;第2个部分拥有者权限;第3个部分所在组权限;第4个部分其他者权限

chmod a+rwx file.py

chown

ps

查看进程

ps -aux | grep uwsgi

查看端口

netstat -lnp|grep 80

kill

杀死进程

kill 进程PID

强制杀死进程

kill 9 进程PID

查找uwsgi位置

find / -name uwsgi

init

进入图形界面

init 5

退出图形界面

init 3

su

切换用户

su root

-t

可以验证语法

httpd -t
