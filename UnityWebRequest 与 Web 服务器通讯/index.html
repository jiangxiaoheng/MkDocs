UnityWebRequest 与 Web 服务器通讯

一，搭建客户端 +http 服务器作为通讯框架

客户端： Unity + UnityWebRequest ---> Http 通讯

服务端：Node.js + express（http 框架）

安装：npm install express

实例：

var express = require('express');    //  通过 require() 导入 Express 模块
var app = express();    // 创建一个 Express 实例
app.listen(6080, function(){console.log('服务器运行中···');});    // 启动服务器时监听端口
app.use(express.static('public'));    // 静态目录，可以不写，默认是 public

执行：node main.js

二，UnityWebRequest 结构详解

http协议，http请求 客户端 ---> web服务器 ---> 发送一个http数据包 ---> http请求 服务器 ---> 收到客户端http请求 ---> 使用http协议解开这个包，拿到数据

UnityWebRequest ---> Unity引擎 ---> Networking

1.构造一个 UnityWebRequest 协议请求：

http get 请求 http post 请求：提交form表单 http put 请求：上传文件 静态的构建方法 ---> 一种协议的对象 ---> 一般使用这种构建方式 UnityWebRequest

2.UnityWebRequest 两个部分：

将客户端数据传到服务器：给服务器上传我们的数据

public UploadHandler uploadHandler {get; set;}

服务器数据 ---> 客户端：获取服务器返回的数据

public DownloadHandler downloadHandler {get; set;}

文件进度：UploadHandler DownloadHander

3.发送一个请求 SendWebRequest 异步

发完后不用等底层完整的发出去，就可以做其他事情，避免卡住游戏线程

三，发送数据到服务器，并存储到数据库

请求 ---> 响应

public void EnterGame(){
    this.StartCoroutine(this.GetUpData());    // 运行该方法
}

// Unity客户端请求： url 携带 get 参数
IEnumerator GetUpData(){   
    UnityWebRequest www = UnityWebRequest.Get("http://127.0.0.1:6080/uploadData?username=ZZY&password=520"); 
    yield return www.SendWebRequest();    // 等待请求完成，发送一个请求 SendWebRequest 异步；
    Debug.Log(www.downloadHandler.text);    // 获取响应的文本；
}

// Web服务端响应：获取 query 的参数，并打印到控制台
app.get('/uploadData', function(req, res) { // 注册一个get请求  /uploadData
    console.log(req.query);   // 打印请求
    res.send('Hello unity!');   // 发送响应
    res.end();  // 结束响应
});

四，资源更新从服务器上拉取游戏资源版本号

public void EnterGame(){
        this.StartCoroutine(this.ReadResVersion()); // 运行该方法
    }

IEnumerator ReadResVersion(){
    UnityWebRequest res = UnityWebRequest.Get("http://127.0.0.1:6080/version.txt");
    yield return res.SendWebRequest();
    Debug.Log(res.downloadHandler.text);
    yield break;    // 必须要有这个，否则会一直卡在这里；
}

五，从服务器上下载资源文件到本地

public void EnterGame(){
        this.StartCoroutine(this.DownloadResFile());    // 运行该方法
    }

IEnumerator DownloadResFile(){
    string url = "http://127.0.0.1:6080/image/logo.png";
    UnityWebRequest res = UnityWebRequest.Get(url);
    yield return res.SendWebRequest();
    byte [] data = res.downloadHandler.data;    // 获取下载的数据；
    string fileName = Application.persistentDataPath + "/zfzlogo.png";  // 私有目录 + 文件名；
    Debug.Log("fileName:" + fileName);
    System.IO.File.WriteAllBytes(fileName, data);   // 写入文件；
    yield break;    // 必须要有这个，否则会一直卡在这里；
}

六，上传文件到服务器

Web 服务端

// 文件系统
var fs = require('fs');
// 注册一个put请求，域名/uploadRes
app.put('/uploadRes', function(req, res) {
    var path = 'public/upload/' + 'logo2.png';
    // fs.openSync() 同步打开文件，参数：文件路径，文件打开模式
    var file = fs.openSync(path, 'w');
    // req.on('data') 监听数据流,每次接收到数据就会触发
    req.on('data', function(data) {
        // fs.write() 异步写入，参数：file文件, data 数据, 0开始写入位置, data.length数据长度, null回调函数
        fs.write(file, data, 0, data.length, null, function() {});
    });
    // req.on('end') 监听end事件
    req.on('end', function() {
        // 发送响应内容到客户端
        res.send('文件上传成功：' + path);
        // 关闭文件
        fs.close(file, function() {});
    });
});

客户端


public void EnterGame(){
    this.StartCoroutine(this.UploadFile()); // 运行该方法
}

IEnumerator UploadFile(){
    string fileName = Application.persistentDataPath + "/zfzlogo.png";  // 私有目录 + 文件名；
    byte[] data = System.IO.File.ReadAllBytes(fileName);          // 读取文件；
    string url = "http://127.0.0.1:6080/uploadFile";
    UnityWebRequest req = UnityWebRequest.Put(url, data);     // 发送请求；
    yield return req.SendWebRequest();
    Debug.Log(req.downloadHandler.text);
    yield break;    // 必须要有这个，否则会一直卡在这里；
}
